Perguntas Frequentes

Tire suas dúvidas sobre o desenvolvimento web e este projeto.

O que é o HTML5?

O HTML5 é a versão mais recente da Linguagem de Marcação de Hipertexto (HTML), que é a linguagem padrão usada para criar e estruturar o conteúdo da web. Ele introduziu novas tags semânticas, suporte nativo para áudio e vídeo, e APIs poderosas para aplicações web modernas.

Para que serve o CSS3?

O CSS3 (Cascading Style Sheets) é a linguagem utilizada para definir a apresentação visual de um documento HTML. Com ele, podemos controlar cores, fontes, layouts (como Grid e Flexbox), animações e garantir que o site seja responsivo em diferentes dispositivos.

Como funciona a propriedade transition?

A propriedade CSS transition permite que você altere os valores das propriedades de forma suave ao longo de um determinado tempo, em vez de ocorrer uma mudança instantânea. É essencial para criar interações mais agradáveis e fluidas para o usuário.

O que são tags semânticas?

Tags semânticas são aquelas que descrevem claramente seu significado tanto para o navegador quanto para o desenvolvedor. Exemplos incluem <header>, <main>, <footer> e <details>. Elas melhoram a acessibilidade e o SEO do site.
